# Tetris Transformer
Tetris Transformer 變形俄羅斯金剛方塊，是一款透過消除方塊來攻擊機器人的遊戲，遊戲中的所有物件皆為 `Windows Form` ，俄羅斯方塊的每個小方塊也都是一個`Windows Form`，因此整個遊戲是在螢幕上進行。

- `shift` can store square
- `space` can place square imminently
